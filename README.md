# Практическая работа 13. Генерация набора заданий с автоматической проверкой

выполнена при помощи Claude Artifacts

**Как работает программа:**

1. `generate_equation()` — сначала выбирается *готовое целое решение* `x`, а затем по нему вычисляется правая часть `c = a*x + b`. Это гарантирует, что ответ всегда целый.
2. `format_equation()` — красиво форматирует уравнение: вместо `3x + -5 = 1` выводит `3x - 5 = 1`.
3. `main()` — основной цикл из 10 задач с проверкой ввода и подсчётом очков.

**Как запустить:**

(_для правильного отображения специальных символов под windows надо запускать в windows terminal_)

```bash
python linear_equations.py
```
